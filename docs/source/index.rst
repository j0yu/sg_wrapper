Getting Help
------------

If you're having trouble or have questions, you can create an issue in the bug tracker (see Issues) and mark it as an ehancement or a proposal.

Issues
------

All issues should reported at the `sg_wrapper bug tracker <http://redmine.mikros.int/projects/{PROJECT}/issues>`_.

Another way is to send an email at <syd@mikrosimage.eu> or <jbro@mikrosimage.eu> (Please also CC <zaf@mikrosimage.eu>). Please provide the following information so we can quickly process your issue:

* Production name
* Terminal output (Copy & pasted from terminal)
* Brief description and/or way to reproducte the issue
